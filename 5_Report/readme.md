# Introduction

In this project we will control the door using switch.if the door opens the led will glow.if the switch is off the door closes and led will off.

# Components

  Atmega328p

  LED bulb

  Connecting Wires

  Resistors

  5V Rail

  Switch

  Proximity sensor
